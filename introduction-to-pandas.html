5 Introduction to Pandas

Pandas is one of the most popular and powerful open-source data analysis and manipulation libraries for Python. This chapter will introduce you to the core features of Pandas, walk you through its data structures, and provide practical examples to help you harness the full potential of this library.

It provides powerful, flexible, and easy-to-use data structures like the DataFrame and Series. With these structures, you can perform a wide variety of operations on datasets, including aggregation, filtering, merging, and time series analysis.

Pandas is commonly used in data science, statistics, finance, and machine learning workflows. It integrates seamlessly with other scientific libraries like NumPy, Matplotlib, and SciPy.

5.1 Installing Pandas

To install Pandas, you can use the following command:

pip install pandas

5.2 The Pandas Data Structures

Pandas is built on two primary data structures: Series and DataFrame. These structures are the foundation of the library and are designed to handle different types of data in a way that is both efficient and easy to manipulate. In this section, we will explore both data structures in detail, looking at their properties, usage, and key functionalities.

5.2.1 Series

A Series is a one-dimensional labeled array capable of holding any data type. It is similar to a Python list or a NumPy array but with an additional feature: an index.

5.2.1.1 Key Characteristics of a Series:

  • One-dimensional: It is a single column of data.
  • Indexing: It has an index, which can be either numerical or label-based, to access the data.
  • Homogeneous data: All elements in a Series must be of the same data type (e.g., integers, floats, strings, etc.).

5.2.1.2 Creating a Series

You can create a Series from a Python list, NumPy array, or a dictionary.

5.2.1.2.1 From a List:
import pandas as pd

data = [10, 20, 30, 40]
series = pd.Series(data)

print(series)
0    10
1    20
2    30
3    40
dtype: int64
5.2.1.2.2 From a Dictionary:
import pandas as pd

data = {'a': 10, 'b': 20, 'c': 30}
series = pd.Series(data)

print(series)
a    10
b    20
c    30
dtype: int64
5.2.1.2.3 Customizing the Index:

You can also specify custom indices when creating a Series.

import pandas as pd

data = [10, 20, 30, 40]
index = ['a', 'b', 'c', 'd']
series = pd.Series(data, index=index)

print(series)
a    10
b    20
c    30
d    40
dtype: int64
5.2.1.2.4 Accessing Elements in a Series

You can access elements in a Series by using the index or the integer location.

By Index: print(series['a']) # Output: 10

By Position (Using iloc) print(series.iloc[2]) # Output: 30

5.2.1.2.5 Operations on Series

Series support element-wise operations. These operations are similar to those in NumPy.

import pandas as pd
data = [10, 20, 30, 40]
index = ['a', 'b', 'c', 'd']
series = pd.Series(data, index=index)
# Adding a scalar value to all elements
series += 5
print(series)
a    15
b    25
c    35
d    45
dtype: int64

5.2.2 DataFrame

A DataFrame is a two-dimensional labeled data structure that consists of rows and columns. It is the most commonly used Pandas structure for handling and analyzing data. You can think of it as a table or spreadsheet where data is organized into rows and columns, and both dimensions are labeled.

5.2.2.1 Key Characteristics of a DataFrame:

  • Two-dimensional: It has both rows and columns.
  • Indexing: A DataFrame has both row and column indices that allow access to its data.
  • Heterogeneous data: Columns can hold different data types (e.g., integers, floats, strings).
  • Data alignment: DataFrames automatically align data for you, even if the data is from different sources.

5.2.2.2 Creating a DataFrame

You can create a DataFrame from a variety of data sources, including lists, dictionaries, or even other DataFrames.

import pandas as pd
data = {
    'Name': ['John', 'Anna', 'Peter'],
    'Age': [28, 24, 35],
    'City': ['New York', 'Paris', 'London']
}
df = pd.DataFrame(data)

print(df)
    Name  Age      City
0   John   28  New York
1   Anna   24     Paris
2  Peter   35    London

From a List of Lists (with Column Names):

import pandas as pd

data = [['John', 28, 'New York'], ['Anna', 24, 'Paris'], ['Peter', 35, 'London']]
df = pd.DataFrame(data, columns=['Name', 'Age', 'City'])

print(df)
    Name  Age      City
0   John   28  New York
1   Anna   24     Paris
2  Peter   35    London

5.2.2.3 Accessing Elements in a DataFrame

You can access data in a DataFrame by row, column, or both.

By Column:

print(df['Name'])
0     John
1     Anna
2    Peter
Name: Name, dtype: object

By Row (Using iloc or loc):

iloc: Access data by integer position.

print(df.iloc[1])  # Access second row
Name     Anna
Age        24
City    Paris
Name: 1, dtype: object

loc: Access data by label/index.

print(df.loc[1])  # Access row with index 1
Name     Anna
Age        24
City    Paris
Name: 1, dtype: object

By Specific Element (Row and Column):

print(df.at[1, 'Name'])  # Access element at row 1, column 'Name'
Anna

5.2.2.4 Operations on DataFrames

DataFrames support a wide range of operations, including element-wise arithmetic, data alignment, and aggregation.

Arithmetic Operations:

# Add 5 to each element in the DataFrame
df['Age'] += 5
print(df)
    Name  Age      City
0   John   33  New York
1   Anna   29     Paris
2  Peter   40    London

Aggregation and Statistical Operations:

print(df['Age'].mean())  # Average age
34.0
print(df['Age'].sum())   # Total age
102

In Pandas, Series and DataFrame are powerful structures that enable you to manipulate and analyze data in an efficient and intuitive way. The Series is ideal for one-dimensional data, while the DataFrame provides a versatile structure for two-dimensional datasets.

5.3 Data Import and Export in Pandas

One of the core functionalities of Pandas is its ability to easily read from and write to a variety of file formats. Whether you’re working with CSV, Excel, SQL databases, JSON, or other formats, Pandas provides powerful tools to import and export data efficiently. This section will cover the most common methods for data import and export in Pandas.

5.3.1 Data import and export

  1. Importing Data
    • CSV Files
    • Excel Files
    • SQL Databases
    • JSON Files
    • Other Formats
  2. Exporting Data
    • CSV Files
    • Excel Files

5.3.2 Importing Data

Pandas makes it easy to import data from a variety of sources, including flat files, databases, and web APIs. Let’s go through some of the most commonly used methods.

5.3.2.1 CSV Files

CSV (Comma-Separated Values) files are one of the most common data formats. Pandas has a built-in function read_csv() to load CSV files into a DataFrame.

df = pd.read_csv('path_to_file.csv')

You can also specify additional parameters like the delimiter, header row, or columns to load.

df = pd.read_csv('data.csv', delimiter=';', header=0)
5.3.2.1.1 Handling Missing Values:

You can handle missing values when importing a CSV by specifying a list of values to be treated as NaN.

df = pd.read_csv('data.csv', na_values=['NA', 'missing', '--'])

5.3.2.2 Excel Files

Pandas provides the read_excel() function to import Excel files. You need to install the openpyxl or xlrd package to work with .xlsx or .xls files.

df = pd.read_excel('path_to_file.xlsx', sheet_name='Sheet1')

You can load multiple sheets from an Excel file by passing a list to sheet_name.

df = pd.read_excel('data.xlsx', sheet_name=['Sheet1', 'Sheet2'])

5.3.2.3 SQL Databases

Pandas makes it easy to work with SQL databases by using the read_sql() and read_sql_query() functions. You can directly load data from a SQL query or a database table into a DataFrame.

import sqlite3
import pandas as pd

conn = sqlite3.connect('example.db')
df = pd.read_sql('SELECT * FROM users', conn)
print(df.head())

5.3.2.4 JSON Files

Pandas can read JSON files with the read_json() function. This is useful for importing hierarchical data.

df = pd.read_json('data.json')
print(df.head())

Pandas automatically handles the conversion of nested JSON structures into DataFrames.

5.3.2.5 Other Formats

Pandas can handle various other file formats, including:

  • HTML: pd.read_html()
  • Parquet: pd.read_parquet()
  • HDF5: pd.read_hdf()
  • Feather: pd.read_feather()
  • Stata: pd.read_stata()
  • SAS: pd.read_sas()

Refer to the Pandas documentation for a full list of supported file formats.

5.3.3 Exporting Data

After performing analysis or transformation on your data, you might need to export it to a different file format. Pandas provides a variety of functions for exporting DataFrames to popular formats like CSV, Excel, and JSON.

5.3.3.1 CSV Files

To export a DataFrame to a CSV file, use the to_csv() function.

df.to_csv('output.csv', index=False)

5.3.3.2 Excel Files

To export a DataFrame to an Excel file, use the to_excel() function. You’ll need to have the openpyxl package installed to write to .xlsx files.

df.to_excel('output.xlsx', index=False)

To write multiple sheets, pass a ExcelWriter object to to_excel().

with pd.ExcelWriter('output.xlsx') as writer:
    df1.to_excel(writer, sheet_name='Sheet1')
    df2.to_excel(writer, sheet_name='Sheet2')

5.3.3.3 SQL Databases

You can export DataFrames back to SQL databases using the to_sql() function. This function writes a DataFrame to an SQL table.

5.3.3.4 Syntax:

df.to_sql('table_name', conn, if_exists='replace', index=False)
  • if_exists='replace': If the table exists, it will be replaced. You can also use append to add data to the existing table.
  • index=False: Avoids writing the DataFrame index as a column in the database.

5.3.3.5 JSON Files

To export data to JSON format, use the to_json() function. This is particularly useful for exporting nested or hierarchical data.

df.to_json('output.json')

You can also control the orientation of the JSON file (e.g., records, index, columns, etc.) using the orient parameter.

df.to_json('output.json', orient='records')

5.3.3.6 Other Formats

Pandas supports exporting DataFrames to other formats as well, including:

  • Parquet: df.to_parquet()
  • HDF5: df.to_hdf()
  • Feather: df.to_feather()
  • Stata: df.to_stata()
  • SAS: df.to_sas()

Pandas provides a powerful and flexible suite of tools for importing and exporting data from a wide range of formats. Whether you’re working with CSV, Excel, SQL databases, JSON, or other file types, Pandas makes it easy to seamlessly move data in and out of your Python environment. By mastering these data import and export techniques, you can efficiently handle your data and integrate Pandas into your larger data processing workflow.

5.4 Data Selection and Filtering in Pandas

One of the key features of Pandas is its powerful and flexible tools for selecting, filtering, and manipulating data within DataFrames and Series. Understanding how to efficiently select and filter data is essential for data analysis tasks. This section covers the most common methods for selecting and filtering data in Pandas.

5.4.1 Data Selection and Filtering

  1. Selecting Data from a DataFrame
    • Selecting Columns
    • Selecting Rows
    • Selecting Specific Data (Row and Column)
  2. Boolean Indexing and Conditional Selection
  3. .loc[] and .iloc[] for Label-based and Position-based Indexing
  4. Filtering Data
    • Filtering Rows Based on Conditions
    • Filtering Columns
    • Filtering with .query()
  5. Modifying Data

5.4.2 Selecting Data from a DataFrame

Pandas provides various ways to select data from a DataFrame. The most common methods are by selecting columns, rows, or both.

5.4.2.1 Selecting Columns

Columns in a DataFrame can be accessed using either dot notation or bracket notation.

5.4.2.1.1 Using Dot Notation:
import pandas as pd

data = {'Name': ['John', 'Anna', 'Peter'], 'Age': [28, 24, 35], 'City': ['New York', 'Paris', 'London']}
df = pd.DataFrame(data)

# Select a single column
age_column = df.Age
print(age_column)
5.4.2.1.2 Using Bracket Notation:
# Select a single column using bracket notation
age_column = df['Age']
print(age_column)

You can also select multiple columns by passing a list of column names.

# Select multiple columns
df_subset = df[['Name', 'City']]
print(df_subset)

5.4.2.2 Selecting Rows

Rows in a DataFrame can be selected using various methods, such as integer-location-based indexing (iloc) or label-based indexing (loc).

5.4.2.2.1 Using iloc[] (Position-based Indexing):
# Select the first row
first_row = df.iloc[0]
print(first_row)

# Select multiple rows (e.g., the first two rows)
first_two_rows = df.iloc[0:2]
print(first_two_rows)
5.4.2.2.2 Using loc[] (Label-based Indexing):
# Select a row by label (index)
row_label = df.loc[0]
print(row_label)

5.4.2.3 Selecting Specific Data (Row and Column)

You can select specific data using both row and column selection simultaneously. This can be done using .loc[] and .iloc[].

5.4.2.3.1 Using .loc[] (Label-based):
# Select data by row label and column label
data_point = df.loc[1, 'Age']
print(data_point)
5.4.2.3.2 Using .iloc[] (Position-based):
# Select data by row position and column position
data_point = df.iloc[1, 1]
print(data_point)

5.4.3 Boolean Indexing and Conditional Selection

Boolean indexing allows you to filter data based on conditions. This technique is useful when you want to select rows or columns that meet specific criteria.

5.4.3.1 Filtering Rows Based on Conditions

You can create boolean conditions to filter rows. For example, if you want to select rows where the “Age” is greater than 30:

# Select rows where Age > 30
filtered_df = df[df['Age'] > 30]
print(filtered_df)

You can use multiple conditions with logical operators (& for “and”, | for “or”).

# Select rows where Age > 30 and City is 'London'
filtered_df = df[(df['Age'] > 30) & (df['City'] == 'London')]
print(filtered_df)

5.4.3.2 Filtering Columns Based on Conditions

You can also filter columns based on conditions. For example, selecting columns where the values in the “Age” column are greater than 25:

# Select columns where Age > 25
filtered_columns = df.loc[:, df['Age'] > 25]
print(filtered_columns)

5.4.4 .loc[] and .iloc[] for Label-based and Position-based Indexing

Both .loc[] and .iloc[] provide powerful ways to select data, with .loc[] being label-based and .iloc[] being position-based.

5.4.4.1 .loc[] (Label-based Indexing)

The .loc[] function is used when you want to select data based on the row and column labels (index).

df.loc[row_indexer, column_indexer]

5.4.4.2 Example:

# Select rows with labels 0 and 1, and columns 'Name' and 'City'
selected_data = df.loc[0:1, ['Name', 'City']]
print(selected_data)

5.4.4.3 .iloc[] (Position-based Indexing)

The .iloc[] function is used when you want to select data based on the integer position of rows and columns.

df.iloc[row_position, column_position]

5.4.4.4 Example:

# Select the first row and first column (position-based)
selected_data = df.iloc[0, 0]
print(selected_data)

5.4.5 Filtering Data

Pandas provides multiple ways to filter data within a DataFrame, either by using conditions, label-based selection, or query functions.

5.4.5.1 Filtering Rows Based on Conditions

You can filter rows in a DataFrame using boolean conditions, as discussed earlier.

# Filter rows where Age > 30
filtered_df = df[df['Age'] > 30]
print(filtered_df)

5.4.5.2 Filtering Columns

You can also filter columns by specifying a list of column names or using conditions based on column values.

# Filter specific columns
selected_columns = df[['Name', 'City']]
print(selected_columns)

You can also select columns dynamically using conditions on another column.

# Select columns where Age > 25
filtered_columns = df.loc[:, df['Age'] > 25]
print(filtered_columns)

5.4.5.3 Filtering with .query()

The .query() method allows for a more readable and efficient way to filter rows based on a condition. The syntax is similar to SQL.

5.4.5.4 Syntax:

df.query('condition')

5.4.5.5 Example:

# Select rows where Age > 30
filtered_df = df.query('Age > 30')
print(filtered_df)

You can also use multiple conditions with and, or, and parentheses.

# Select rows where Age > 25 and City is 'London'
filtered_df = df.query('Age > 25 and City == "London"')
print(filtered_df)

5.4.6 Modifying Data

Once you have selected and filtered your data, you can also modify it in place.

5.4.6.1 Modifying Columns

You can directly assign new values to a column.

# Modify a column by assigning new values
df['Age'] = df['Age'] + 1
print(df)

5.4.6.2 Modifying Specific Data Points

You can modify specific data points using .loc[] or .iloc[].

5.4.6.3 Using .loc[]:

# Modify specific value by label
df.loc[0, 'Age'] = 30
print(df)

5.4.6.4 Using .iloc[]:

# Modify specific value by position
df.iloc[0, 1] = 30
print(df)

Pandas provides flexible and powerful methods for selecting, filtering, and modifying data within DataFrames. Whether you’re using label-based or position-based indexing, boolean indexing, or the .query() method, Pandas makes it easy to extract and manipulate data based on specific criteria. Mastering these techniques is essential for efficient data analysis and data wrangling.

5.5 How NumPy Forms a Basis for Pandas

NumPy is a foundational library in Python for numerical computing, and it plays a critical role in enabling the functionality of Pandas. Pandas, a high-level data manipulation library, is built on top of NumPy and leverages its efficient array operations. Below, we explore how NumPy supports Pandas and why understanding NumPy is essential for using Pandas effectively.

5.5.1 Data Structures in Pandas Are Built on NumPy Arrays

  • Pandas’ core data structures, such as Series and DataFrame, are built upon NumPy arrays (ndarray).
  • The underlying NumPy arrays ensure fast and efficient operations in Pandas by enabling vectorized computations.
import numpy as np
import pandas as pd

# Create a NumPy array
data = np.array([1, 2, 3, 4])

# Use the array in a Pandas Series
series = pd.Series(data)
print(series)

5.5.2 Efficient Numerical Operations

  • NumPy provides the numerical backbone for Pandas. Functions like mean, sum, and std in Pandas are directly powered by NumPy’s implementations.
  • This ensures high performance and compatibility with large datasets.
# NumPy and Pandas numerical operations
arr = np.array([10, 20, 30, 40])
data = pd.Series(arr)

# Using Pandas methods (powered by NumPy)
print(data.mean())  # Mean value
print(data.sum())   # Sum of values

5.5.3 Indexing and Slicing

  • NumPy’s array slicing and indexing capabilities are extended in Pandas, allowing for more powerful and intuitive data manipulation.
# NumPy indexing
arr = np.array([1, 2, 3, 4, 5])
print(arr[1:4])  # Slicing in NumPy

# Pandas indexing
data = pd.Series(arr)
print(data[1:4])  # Slicing in Pandas

5.5.4 Handling Missing Data

  • While NumPy arrays do not inherently support missing values, Pandas introduces NaN (Not a Number) to handle them effectively.
  • NumPy’s np.nan is used internally in Pandas to represent missing data.
# NumPy with NaN
arr = np.array([1, 2, np.nan, 4])

# Pandas with NaN
data = pd.Series(arr)
print(data.isna())  # Check for missing values

5.5.5 Advanced Array Operations

  • NumPy’s advanced operations, such as broadcasting, linear algebra, and element-wise computations, enhance Pandas’ ability to manipulate data efficiently.
# Broadcasting in NumPy
arr = np.array([1, 2, 3])
print(arr * 2)  # Element-wise multiplication

# Pandas broadcasting
data = pd.Series(arr)
print(data * 2)

5.5.6 Interoperability Between NumPy and Pandas

  • NumPy arrays and Pandas objects can be seamlessly converted between each other, ensuring compatibility and ease of use.
# NumPy array to Pandas Series
arr = np.array([1, 2, 3])
data = pd.Series(arr)

# Pandas Series to NumPy array
back_to_array = data.to_numpy()
print(back_to_array)
